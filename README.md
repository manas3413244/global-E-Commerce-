# Global E-Commerce Dashboard

## Project Overview
This project is an Excel Dashboard based on Global E-Commerce Sales Data.

The dashboard helps analyze:
- Total Sales
- Profit
- Orders
- Customer Segments
- Regional Performance
- Top Products
- Sales Trends

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Conditional Formatting

## Dashboard Features
- Interactive Slicers
- KPI Cards
- Sales Analysis
- Profit Analysis
- Region-wise Performance
- Monthly Trends

## Insights
- Technology category generated highest sales.
- Some regions have high sales but low profit.
- Consumer segment contributed maximum revenue.

## Conclusion
This dashboard helps businesses track sales performance and make better business decisions.
